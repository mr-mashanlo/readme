Обо мне

Я frontend-разработчик с общим опытом более 4 лет: 2 года работал в компании, ещё 2 — на фрилансе. Ранее был частью команды в Smirnov School, где занимался разработкой лендингов для курсов, акций, мероприятий и распродаж. Работал с гибкими системами скидок и следил, чтобы всё выглядело аккуратно и работало без сбоев.

Умею работать с React.js, WordPress и Node.js, создаю качественную адаптивную вёрстку по БЭМ, знаю SASS и PostCSS, собираю проекты с помощью Gulp под разные задачи. Также умею пользоваться Figma, Photoshop и Illustrator, могу собрать простой и аккуратный веб-дизайн. Стараюсь писать чистый и понятный код, чтобы продукт был удобен для пользователя, а поддержка — простой для команды.

Проекты

  • Многостраничный сайт в тёмной теме с фильтрацией и слайдерами

    Современный сайт из 6 страниц, оформленный в тёмной цветовой палитре с акцентом на визуальный стиль. Реализованы анимированные слайдеры, интерактивная фильтрация контента и логичная семантическая структура. Все страницы адаптивны, протестированы на доступность (WCAG) и валидны по W3C. Оптимизирован для быстрой загрузки. Исходный код доступен на GitHub.

  • Яркий многостраничный сайт с галереей masonry и акцентом на доступность

    Лёгкий и выразительный сайт на 3 страницы, выполненный в яркой цветовой гамме. Вся вёрстка соответствует стандартам HTML5 и WCAG: элементы корректно читаются скринридерами, навигация доступна с клавиатуры, использована семантическая структура и ARIA-атрибуты. Проект протестирован с помощью IBM Equal Access Accessibility Checker. Исходный код доступен на GitHub.

  • React-приложение для генерации страниц с шэринг-ссылками

    Приложение позволяет пользователям создавать персональные страницы с информацией (например, контакты, ссылки на соцсети, описание) и делиться ими через уникальную ссылку. Использована клиент-серверная архитектура, поддерживается динамическая генерация метатегов для social sharing. Исходный код доступен на GitHub. Вы можете посмотреть демо-версию проекта.

  • Remix.js приложение интернет-магазина с личным кабинетом и фильтрацией

    Полноценное приложение интернет-магазина, реализованное на Remix.js. Поддерживает авторизацию, личный кабинет пользователя, добавление товаров в корзину, фильтрацию по категориям и цене. Исходный код доступен на GitHub (Frontend) и GitHub (Backend). Вы можете посмотреть демо-версию проекта.
